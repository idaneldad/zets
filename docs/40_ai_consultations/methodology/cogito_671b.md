## Methodology Improvement
Implement weighted wisdom endorsement:
- Each model assigns confidence scores (0-100) to each concrete claim they make
- Synthesis prioritizes higher-confidence claims
- Enable dynamic claim tracing: "Why did model X say this? What alternatives were discarded?"
- Patterns in confidence scores become forensic tools

This surfaces meta-level knowledge about knowledge (which model is confident about what and why) rather than forcing boolean consensus/conflict.
